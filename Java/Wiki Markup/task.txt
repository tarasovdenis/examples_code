дан текст на простом wiki-подобном языке разметки. Текст состоит из набора строк. Сначала необходимо для всех строк убрать лидирующие, хвостовые и идущие подряд пробелы, игнорировать пустые строки, а также строки, состоящие только из пробельных символов.
Далее необходимо выделить элементы списков среди строк текста. Каждая строка может быть либо элементом списка, либо абзацем. Элементы списка из текста выделяются по принципу: две или более подряд идущие строки, которые начинаются с символа «*» обозначают список. Остальные части текста — абзацы. Несколько подряд идущих строк, не являющихся элементами списка, соедините в один абзац через пробел. Вложенных списков, спецсимволов в тексте нет.

Из строк-элементов списка необходимо удалить начальный символ «*» и последующие за ним пробелы. Возможно, что после этого строка станет пустой, в этом случае она все равно считается элементом списка.

При выводе элемент списка обозначайте тегами <li> и </li>. Сам список окружайте тегами <ul> и </ul> (эти теги должны выводиться на отдельных строках). Абзац окружайте тегами <p> и </p>.
